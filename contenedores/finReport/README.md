# Importante

- En este directorio se debe agregar archivo con nombre postgres.env

- Reemplazar <> por los valores que se requiere cambiar

- Si estas en un linux, antes de ejecutar el .sh

chmod 600 postgres.env
chmod +x run_postgres.sh


## Ejemplo de archivo postgres.env


# ==========================================
# PostgreSQL - FinReport
# ==========================================
POSTGRES_DB=<nombre de la instancia>
POSTGRES_USER=<usuario>
POSTGRES_PASSWORD=<Password>
